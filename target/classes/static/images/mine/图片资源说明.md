# 图片资源说明

## 当前已有图片
- `avatar-default.jpg` - 默认头像
- `header-bg.jpg` - 默认背景图片

## 需要添加的图片

### 头像图片（可选）
以下头像图片需要您自己准备，放在此目录下：
- `avatar1.jpg` - 可爱风格 1
- `avatar2.jpg` - 可爱风格 2
- `avatar3.jpg` - 清新风格

建议尺寸：200x200 像素或更大
格式：JPG 或 PNG

### 背景图片（可选）
以下背景图片需要您自己准备，放在此目录下：
- `bg1.jpg` - 清新绿色背景
- `bg2.jpg` - 温暖棕色背景
- `bg3.jpg` - 简约米白背景

建议尺寸：800x600 像素或更大
格式：JPG 或 PNG

## 如何使用

1. 将准备好的图片放入此目录
2. 刷新页面即可在个人信息编辑页面看到新选项
3. 用户可以选择自己喜欢的头像和背景

## 数据库存储

用户选择的头像和背景图片路径会存储到数据库的 `user` 表中：
- `avatar` 字段：存储头像路径
- `header_background` 字段：存储背景图片路径

路径格式示例：
- `../images/mine/avatar-default.jpg`
- `../images/mine/bg1.jpg`
