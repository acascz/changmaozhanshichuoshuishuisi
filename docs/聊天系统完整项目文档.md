## 🚀 快速开始

### 1. 数据库初始化

按顺序执行以下 SQL 脚本：

```bash
# 第一阶段：基础架构
mysql -u root -p your_database < src/main/resources/sql/chat_system_schema.sql

# 第五阶段：群聊与客服
mysql -u root -p your_database < src/main/resources/sql/chat_stage5_group_service.sql

# 第六阶段：优化与完善
mysql -u root -p your_database < src/main/resources/sql/chat_stage6_optimization.sql
```

### 2. 启动 Redis

```bash
# 确保 Redis 在以下路径
cd redis
./redis-server.exe redis.windows.conf
```

或者使用 Maven 自动启动（已配置）：
```bash
mvn spring-boot:run
```

### 3. 启动应用

```bash
mvn spring-boot:run
```

应用将在 `http://localhost:8080` 启动

## 📋 功能演示

### 1. 基础消息收发

1. 访问 `http://localhost:8080/chat-complete.html`
2. 选择一个会话
3. 在输入框输入消息
4. 点击"发送"或按回车
5. 消息实时显示

### 2. 创建群聊

1. 点击右侧"创建群聊"按钮
2. 输入群名称
3. 输入成员 ID 列表（用逗号分隔）
4. 点击"创建"
5. 群聊自动出现在会话列表

### 3. 加入群聊

1. 点击右侧"加入群聊"按钮
2. 输入群聊 ID
3. 点击"加入"
4. 成功加入后可以在会话列表看到该群

### 4. 群聊管理

**更新群公告**：
- 选择一个群聊
- 点击右侧"更新群公告"
- 输入新的公告内容

**转让群主**：
- 仅群主可以操作
- 选择要转让的成员
- 确认后群主变更为该成员

**移除成员**：
- 群主或管理员可操作
- 选择要移除的成员
- 确认后成员被移除

**设置管理员**：
- 仅群主可操作
- 选择要设置的成员
- 确认后成员成为管理员

### 5. 客服系统

**用户请求客服**：
1. 点击右侧"请求客服"按钮
2. 系统自动分配客服
3. 显示排队人数
4. 分配成功后开始聊天

**客服上线/下线**：
1. 点击"客服上线"
2. 输入客服 ID
3. 上线后可以接收用户咨询

**客服评价**：
1. 在客服会话中点击"评价客服"
2. 输入评分（1-5 星）
3. 可选填写评价内容
4. 提交评价

### 6. 性能监控

右侧面板实时显示：
- 缓存命中次数
- 缓存未命中次数
- 缓存命中率
- 操作日志

## 🔧 API 接口文档

### 群聊 API

```
POST /api/chat/group/create
  参数：creatorId, groupName, groupAvatar(可选), memberIds(可选)
  功能：创建群聊

POST /api/chat/group/join
  参数：groupId, userId
  功能：加入群聊

POST /api/chat/group/leave
  参数：groupId, userId
  功能：退出群聊

POST /api/chat/group/dismiss
  参数：groupId, operatorId
  功能：解散群聊

GET /api/chat/group/list
  参数：userId
  功能：获取用户的群聊列表

GET /api/chat/group/detail
  参数：groupId
  功能：获取群聊详情

POST /api/chat/group/notice/update
  参数：groupId, userId, notice
  功能：更新群公告

POST /api/chat/group/transfer
  参数：groupId, currentOwnerId, newOwnerId
  功能：转让群主

POST /api/chat/group/member/remove
  参数：groupId, operatorId, memberId
  功能：移除群成员

POST /api/chat/group/admin/set
  参数：groupId, operatorId, memberId
  功能：设置管理员
```

### 客服系统 API

```
POST /api/chat/service/request
  参数：userId, question(可选)
  功能：用户请求客服

POST /api/chat/service/online
  参数：serviceId
  功能：客服上线

POST /api/chat/service/offline
  参数：serviceId
  功能：客服下线

POST /api/chat/service/transfer
  参数：sessionId, fromServiceId, toServiceId
  功能：转接客服

POST /api/chat/service/end
  参数：sessionId, userId
  功能：结束会话

GET /api/chat/service/status
  功能：获取客服状态

POST /api/chat/service/rating/submit
  参数：sessionId, userId, serviceId, rating, comment(可选)
  功能：提交评价

GET /api/chat/service/rating/stats
  参数：serviceId
  功能：获取评分统计

GET /api/chat/service/rating/list
  参数：serviceId
  功能：获取评价列表
```

### 消息 API

```
GET /api/chat/messages/{sessionId}
  功能：获取会话消息列表

GET /api/chat/messages/detail/{messageId}
  功能：获取消息详情

POST /api/chat/messages/{messageId}/read
  功能：标记已读

POST /api/chat/messages/{messageId}/recall
  功能：撤回消息

DELETE /api/chat/messages/{messageId}
  功能：删除消息
```

## 📈 性能优化

### Redis 缓存策略

**缓存内容**：
- 群聊详情（30 分钟过期）
- 用户群聊列表（10 分钟过期）

**缓存命中率**：
- 预期：80%+
- 实际：查看右侧性能监控面板

**缓存清除**：
- 群聊信息变更时自动清除
- 成员变动时自动清除

### 数据库索引

**已优化索引**：
- `chat_group.member_count`
- `chat_group.status_created`
- `chat_session_member.role_join`
