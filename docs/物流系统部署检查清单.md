### 5. 部署启动

#### 5.1 直接运行 JAR 包
- [ ] 上传 JAR 包到服务器
```bash
scp target/mung-bean-cake-mall-1.0.0.jar user@server:/opt/logistics/
```
- [ ] 上传配置文件
```bash
scp src/main/resources/application-prod.yml user@server:/opt/logistics/
```
- [ ] 创建启动脚本
```bash
vim /opt/logistics/start.sh
```

**启动脚本内容**：
```bash
#!/bin/bash
APP_NAME=mung-bean-cake-mall-1.0.0.jar
APP_DIR=/opt/logistics

cd $APP_DIR

# JVM 参数
JAVA_OPTS="-Xms2g -Xmx4g"
JAVA_OPTS="$JAVA_OPTS -XX:+UseG1GC -XX:MaxGCPauseMillis=200"
JAVA_OPTS="$JAVA_OPTS -Dspring.profiles.active=prod"

# 启动应用
nohup java $JAVA_OPTS -jar $APP_NAME > app.log 2>&1 &

# 打印进程 ID
echo "应用已启动，PID: $!"
```

- [ ] 赋予执行权限
```bash
chmod +x /opt/logistics/start.sh
```
- [ ] 启动应用
```bash
/opt/logistics/start.sh
```
- [ ] 查看日志
```bash
tail -f /opt/logistics/app.log
```

#### 5.2 使用 Systemd 管理（推荐）
- [ ] 创建服务文件
```bash
vim /etc/systemd/system/logistics.service
```

**服务文件内容**：
```ini
[Unit]
Description=Mung Bean Cake Mall Logistics System
After=syslog.target network.target

[Service]
User=root
WorkingDirectory=/opt/logistics
ExecStart=/usr/bin/java -Xms2g -Xmx4g -jar /opt/logistics/mung-bean-cake-mall-1.0.0.jar --spring.profiles.active=prod
SuccessExitStatus=143
Restart=on-failure
RestartSec=10

StandardOutput=journal
StandardError=journal
SyslogIdentifier=logistics

[Install]
WantedBy=multi-user.target
```

- [ ] 重载 systemd 配置
```bash
systemctl daemon-reload
```
- [ ] 启动服务
```bash
systemctl start logistics
```
- [ ] 设置开机自启
```bash
systemctl enable logistics
```
- [ ] 查看状态
```bash
systemctl status logistics
```
- [ ] 查看日志
```bash
journalctl -u logistics -f
```

### 8. Nginx 配置（可选）

#### 8.1 反向代理配置
- [ ] 创建 Nginx 配置文件
```bash
vim /etc/nginx/conf.d/logistics.conf
```

**配置内容**：
```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    # 前端静态文件
    location / {
        root /var/www/logistics;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
    
    # 后端 API 代理
    location /api/ {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_connect_timeout 60s;
        proxy_read_timeout 60s;
    }
    
    # 静态资源缓存
    location ~* \.(jpg|jpeg|png|gif|ico|css|js)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}
```

- [ ] 测试 Nginx 配置
```bash
nginx -t
```
- [ ] 重载 Nginx
```bash

```
