## 1. 微信登录

### 📌 所需资质
- **企业营业执照**（个体工商户或公司）
- **对公账户**（用于微信认证，300 元/年认证费）
- **已备案的域名**（必须与公众号/开放平台账号绑定）

### 🔑 需要准备的密钥

| 密钥名称 | 说明 | 获取方式 |
|---------|------|---------|
| `AppID` | 微信开放平台应用的唯一标识 | 登录 [微信开放平台](https://open.weixin.qq.com) → 管理中心 → 网站应用 → 查看 AppID |
| `AppSecret` | 微信开放平台应用的密钥 | 同上，点击"查看"按钮获取（需要管理员扫码） |

### 📝 申请步骤

1. **注册微信开放平台账号**
   - 访问 https://open.weixin.qq.com
   - 注册企业账号并完成邮箱激活

2. **开发者资质认证**
   - 提交企业营业执照
   - 支付 300 元认证费（每年）
   - 等待 1-3 个工作日审核

3. **创建网站应用**
   - 进入"管理中心" → "网站应用"
   - 点击"创建网站应用"
   - 填写应用信息（名称、描述、图标等）
   - 填写网站信息（域名、回调地址等）
   - 提交审核（1-3 个工作日）

4. **获取密钥**
   - 审核通过后，在网站应用详情页查看 AppID
   - 点击"查看"AppSecret（需要管理员微信扫码）

### 🔗 回调地址配置

```
回调地址：https://yourdomain.com/api/auth/wechat/callback
```

**注意：** 
- 域名必须已备案
- 必须使用 HTTPS 协议
- 本地测试可以使用内网穿透工具（如 natapp、ngrok）

### 📚 相关 API 文档

- [微信开放平台 - 网站应用微信登录开发文档](https://developers.weixin.qq.com/doc/oplatform/Website_App/WeChat_Login/Wechat_Login.html)

## 2. QQ 登录

### 📌 所需资质
- **企业营业执照**（或个人开发者）
- **已备案的域名**（个人开发者也需要）
- **网站 ICP 备案**

### 🔑 需要准备的密钥

| 密钥名称 | 说明 | 获取方式 |
|---------|------|---------|
| `AppID` | QQ 互联应用的唯一标识 | 登录 [QQ 互联](https://connect.qq.com) → 应用管理 → 网站应用 → 查看 AppID |
| `AppKey` | QQ 互联应用的密钥 | 同上，在应用详情页查看 |

### 📝 申请步骤

1. **注册 QQ 互联账号**
   - 访问 https://connect.qq.com
   - 使用 QQ 号登录并注册为开发者

2. **开发者认证**
   - 个人开发者：上传身份证正反面
   - 企业开发者：上传营业执照
   - 等待 1-3 个工作日审核

3. **创建网站应用**
   - 进入"应用管理" → "网站应用"
   - 点击"创建网站应用"
   - 填写应用信息
   - 填写网站信息（域名、回调地址等）
   - 提交审核（1-3 个工作日）

4. **获取密钥**
   - 审核通过后，在应用详情页查看 AppID 和 AppKey

### 🔗 回调地址配置

```
回调地址：https://yourdomain.com/api/auth/qq/callback
```

**注意：**
- 域名必须已备案
- 必须使用 HTTPS 协议

### 📚 相关 API 文档

## 3. 邮箱 SMTP

### 📌 所需资质
- **个人邮箱**（推荐使用 QQ 邮箱、163 邮箱、Gmail 等）
- **无需企业资质**

### 🔑 需要准备的密钥

| 密钥名称 | 说明 | 获取方式 |
|---------|------|---------|
| `SMTP 服务器地址` | 邮箱的 SMTP 服务器地址 | 邮箱设置 → POP3/SMTP/IMAP 设置 |
| `SMTP 端口` | SMTP 服务器端口 | 通常为 587（TLS）或 465（SSL） |
| `邮箱账号` | 用于发送邮件的邮箱地址 | 你的邮箱地址 |
| `SMTP 授权码` | 邮箱的 SMTP 授权码（不是邮箱密码） | 邮箱设置 → 开启 SMTP 服务 → 获取授权码 |

### 📝 常见邮箱 SMTP 配置

#### QQ 邮箱
```
SMTP 服务器：smtp.qq.com
端口：587（TLS）或 465（SSL）
获取授权码：邮箱设置 → 账户 → 开启 SMTP 服务 → 生成授权码
```

#### 163 邮箱
```
SMTP 服务器：smtp.163.com
端口：587（TLS）或 465（SSL）
获取授权码：邮箱设置 → POP3/SMTP/IMAP → 开启 SMTP 服务 → 获取授权码
```

#### Gmail
```
SMTP 服务器：smtp.gmail.com
端口：587（TLS）或 465（SSL）
获取授权码：Google 账号设置 → 安全性 → 应用专用密码
```

### 📚 配置示例

```yaml
third-party:
  email:
    host: "smtp.qq.com"
    username: "your-email@qq.com"
    password: "your_smtp_auth_code"  # 这是授权码，不是邮箱密码
    port: 587
```

## 4. 微信支付

### 📌 所需资质
- **企业营业执照**（个体工商户或公司）
- **对公账户**
- **已备案的域名**
- **微信商户号**（需要申请）

### 🔑 需要准备的密钥

| 密钥名称 | 说明 | 获取方式 |
|---------|------|---------|
| `AppID` | 微信公众号/小程序的 AppID | 微信公众平台 → 开发设置 |
| `MchID` | 微信支付商户号 | 微信商户平台 → 账户中心 |
| `API 密钥 (v2)` | 微信支付 API v2 版本密钥 | 微信商户平台 → 账户中心 → API 安全 → 设置 API 密钥 |
| `API 密钥 (v3)` | 微信支付 API v3 版本密钥 | 微信商户平台 → 账户中心 → API 安全 → APIv3 密钥 |
| `商户私钥` | 商户 API 私钥（用于退款等敏感操作） | 微信商户平台 → 账户中心 → API 安全 → 申请 API 证书 |
| `商户证书序列号` | 商户 API 证书序列号 | 下载证书后查看 |

### 📝 申请步骤

1. **注册微信公众号/小程序**
   - 访问 https://mp.weixin.qq.com
   - 注册并完成认证（300 元/年）

2. **申请微信商户号**
   - 访问 https://pay.weixin.qq.com
   - 注册并提交资料（营业执照、法人身份证、对公账户等）
   - 等待 1-5 个工作日审核
   - 审核通过后，完成账户验证（打款验证）

3. **绑定 AppID 和商户号**
   - 登录微信商户平台
   - 产品中心 → AppID 账号管理 → 绑定 AppID

4. **配置 API 密钥**
   - 账户中心 → API 安全 → 设置 API 密钥
   - 下载并安装 API 证书

### 🔗 回调地址配置

```
支付结果通知地址：https://yourdomain.com/api/payment/wechat/notify
```

### 📚 相关 API 文档

- [微信支付开发文档](https://pay.weixin.qq.com/wiki/doc/apiv3/index.shtml)
- [微信支付 API v3 接口文档](https://pay.weixin.qq.com/wiki/doc/apiv3/open/pay/chapter2.shtml)

## 5. 支付宝支付

### 📌 所需资质
- **企业营业执照**（个体工商户或公司）
- **对公账户**
- **已备案的域名**
- **支付宝商户号**

### 🔑 需要准备的密钥

| 密钥名称 | 说明 | 获取方式 |
|---------|------|---------|
| `AppID` | 支付宝应用的唯一标识 | 支付宝开放平台 → 应用详情 |
| `应用私钥` | 应用私钥（用于签名） | 使用支付宝密钥生成工具生成 |
| `支付宝公钥` | 支付宝公钥（用于验证签名） | 上传应用公钥后，支付宝返回公钥 |
| `支付宝根证书` | 支付宝根证书 | 支付宝开放平台下载 |

### 📝 申请步骤

1. **注册支付宝开放平台账号**
   - 访问 https://open.alipay.com
   - 使用支付宝账号登录并完善信息

2. **创建应用**
   - 进入"控制台" → "网页/移动应用"
   - 创建应用并填写信息
   - 添加"电脑网站支付"或"手机网站支付"功能
   - 提交审核（1-3 个工作日）

3. **生成密钥**
   - 下载"支付宝密钥生成工具"
   - 生成应用私钥和公钥
   - 上传应用公钥到支付宝开放平台
   - 获取支付宝公钥

4. **配置回调地址**
   - 应用详情 → 开发信息 → 设置回调地址

### 🔗 回调地址配置

```
异步通知地址：https://yourdomain.com/api/payment/alipay/notify
同步跳转地址：https://yourdomain.com/api/payment/alipay/return
```

### 📚 相关 API 文档

- [支付宝开放平台](https://opendocs.alipay.com/)
- [电脑网站支付文档](https://opendocs.alipay.com/open/194/10549)

## 6. 短信服务

### 📌 所需资质
- **企业营业执照**（部分服务商支持个人）
- **已备案的域名**（部分需要）

### 🔑 需要准备的密钥（以阿里云为例）

| 密钥名称 | 说明 | 获取方式 |
|---------|------|---------|
| `AccessKey ID` | 阿里云访问密钥 ID | 阿里云控制台 → AccessKey 管理 |
| `AccessKey Secret` | 阿里云访问密钥 Secret | 同上 |
| `短信签名` | 短信签名（如【中豆商城】） | 短信服务控制台 → 签名管理 → 添加签名 |
| `短信模板 CODE` | 短信模板 CODE | 短信服务控制台 → 模板管理 → 添加模板 |

### 📝 申请步骤（阿里云）

1. **注册阿里云账号**
   - 访问 https://www.aliyun.com
   - 注册并完成实名认证

2. **开通短信服务**
   - 进入短信服务控制台
   - 开通短信服务（按量付费）

3. **添加签名**
   - 签名管理 → 添加签名
   - 提交资质证明（营业执照等）
   - 等待审核（1-3 个工作日）

4. **添加模板**
   - 模板管理 → 添加模板
   - 模板内容示例：`您的验证码是${code}，10 分钟内有效`
   - 等待审核（1-3 个工作日）

5. **获取密钥**
   - AccessKey 管理 → 创建 AccessKey

### 💰 参考价格

- 阿里云短信：约 0.045 元/条（国内）
- 腾讯云短信：约 0.045 元/条（国内）
- 七牛云短信：约 0.045 元/条（国内）

### 📚 相关 API 文档

- [阿里云短信服务](https://help.aliyun.com/product/44282.html)
- [腾讯云短信服务](https://cloud.tencent.com/document/product/382)

## 7. 配置位置

### 📄 配置文件路径

```
d:\A\changmaozhanshichuoshuishuisi\src\main\resources\application.yml
```

### 📝 配置示例

打开 `application.yml`，找到以下部分并填写真实密钥：

```yaml
# 第三方登录配置
third-party:
  # 微信开放平台配置
  wechat:
    appid: "wx1234567890abcdef"      # 替换为你的微信 AppID
    secret: "your_wechat_secret"      # 替换为你的微信 AppSecret
    redirect-uri: "http://localhost:8080/api/auth/wechat/callback"
  
  # QQ 互联配置
  qq:
    appid: "1234567890"               # 替换为你的 QQ AppID
    appkey: "your_qq_appkey"          # 替换为你的 QQ AppKey
    redirect-uri: "http://localhost:8080/api/auth/qq/callback"
  
  # 邮箱 SMTP 配置
  email:
    host: "smtp.qq.com"
    username: "your-email@qq.com"     # 替换为你的邮箱
    password: "your_smtp_auth_code"   # 替换为 SMTP 授权码
    port: 587
```

### ⚠️ 注意事项

1. **密钥安全**
   - 不要将密钥提交到 Git 仓库
   - 生产环境使用环境变量或配置中心
   - 定期更换密钥

2. **本地测试**
   - 微信/QQ 登录需要公网域名，本地测试可使用内网穿透工具
   - 邮箱 SMTP 可以直接使用

3. **审核时间**
   - 微信开放平台：1-3 个工作日
   - QQ 互联：1-3 个工作日
   - 支付接口：1-5 个工作日
   - 短信签名/模板：1-3 个工作日

4. **费用**
   - 微信开放平台认证：300 元/年
   - 微信公众号认证：300 元/年
   - 微信支付：无年费，按交易费率收费（约 0.6%）
   - 支付宝：无年费，按交易费率收费（约 0.6%）
   - 短信服务：约 0.045 元/条
