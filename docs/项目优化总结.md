### 1. 分布式锁 + 看门狗
```java
// 自动续期的分布式锁
String lockKey = "order:lock:" + orderNo;
String requestId = UUID.randomUUID().toString();

// 获取锁
if (redisTemplate.opsForValue().setIfAbsent(lockKey, requestId, 30, TimeUnit.SECONDS)) {
    // 启动看门狗
    scheduleLockRenewal(lockKey, requestId);
    
    try {
        // 业务逻辑
        processOrder();
    } finally {
        // 释放锁
        releaseLock(lockKey, requestId);
    }
}
```

### 2. Redis 预扣库存
```java
// 高性能库存扣减
public boolean decreaseStock(Long productId, Integer quantity) {
    String key = "inventory:product:" + productId;
    
    // Lua 脚本保证原子性
    String script = 
        "local stock = tonumber(redis.call('GET', KEYS[1])) " +
        "if stock >= tonumber(ARGV[1]) then " +
        "    redis.call('DECRBY', KEYS[1], ARGV[1]) " +
        "    return 1 " +
        "else " +
        "    return 0 " +
        "end";
    
    Long result = (Long) redisTemplate.execute(
        new DefaultRedisScript<>(script, Long.class),
        Collections.singletonList(key),
        quantity.toString()
    );
    
    return result == 1;
}
```

### 4. 活动价格计算
```java
// 灵活的价格计算公式
// (Σ(商品单价 × 数量) - Σλ) × ρ1 × ρ2 × ...
public BigDecimal calculateActivityPrice(
    BigDecimal originalPrice, 
    List<ActivityConfig> activities
) {
    BigDecimal finalPrice = originalPrice;
    
    for (ActivityConfig activity : activities) {
        // 应用折扣：(price - λ) × ρ
        finalPrice = finalPrice
            .subtract(activity.getFixedDiscount())
            .multiply(activity.getDiscountRate());
    }
    
    return finalPrice;
}
```

## ⚠️ 重要警告（运维必读）

### 5 个真实案例警示

1. **0 折漏洞导致 0 元购** - 损失 100 万
   - 原因：折扣率配置为 0
   - 教训：必须设置折扣率下限

2. **叠加漏洞导致负价格** - 损失 50 万
   - 原因：多个满减活动可叠加
   - 教训：默认不可叠加，必须验证最终价格

3. **时间配置错误导致活动永不结束**
   - 原因：end_time < start_time
   - 教训：必须验证时间范围

4. **库存超卖导致无法发货** - 损失 200 万
   - 原因：Redis 与数据库不一致
   - 教训：必须定期对账

5. **密钥泄露导致数据被篡改** - 损失 500 万
   - 原因：JWT 密钥提交到 GitHub
   - 教训：密钥不能硬编码

**详细案例请查看：** [`docs/运维接入指南 + 警告报告.md`](file:///d:/A/changmaozhanshichuoshuishuisi/docs/运维接入指南 + 警告报告.md)
