# 🎉 全新聊天系统使用指南

## 📋 项目概述

这是一个**完整的聊天系统前端界面**，整合了六个阶段的所有功能：
- ✅ 第一阶段：基础聊天架构
- ✅ 第二阶段：消息加密与安全
- ✅ 第三阶段：多种消息类型
- ✅ 第四阶段：AI 智能助手
- ✅ 第五阶段：群聊与客服系统
- ✅ 第六阶段：性能优化

---

## 🚀 快速开始

### 1. 确保后端服务已启动

```bash
# 在终端 5 中应该已经运行
mvn spring-boot:run
```

应用将在 `http://localhost:8080` 启动

### 2. 访问聊天界面

**主聊天页面**：
```
http://localhost:8080/chat-full.html
```

**备用测试页面**：
```
http://localhost:8080/chat-complete.html
```

## 💡 功能演示

### 1. 好友聊天

**添加好友**：
1. 点击底部"加好友"按钮
2. 输入好友 ID
3. 填写验证消息（可选）
4. 点击"发送申请"

**开始聊天**：
1. 在会话列表中点击好友
2. 在输入框输入消息
3. 按 Enter 或点击"发送"

### 2. 创建群聊

1. 点击底部"建群"按钮
2. 输入群聊名称
3. 勾选要邀请的好友
4. 点击"创建"
5. 群聊自动出现在会话列表

### 3. 联系客服

1. 点击底部"客服"按钮
2. 选择服务类型（订单、退款、商品等）
3. 描述问题
4. 点击"开始咨询"
5. 系统自动分配客服

### 4. AI 助手

1. 点击底部"AI"按钮
2. 自动切换到 AI 聊天窗口
3. 输入问题
4. AI 智能回复

### 5. 发送表情

1. 点击表情按钮 😊
2. 选择表情
3. 表情自动插入输入框
4. 发送

### 6. 搜索聊天

1. 在搜索框输入关键词
2. 实时过滤会话列表
3. 清空搜索框恢复全部显示

## 📝 API 接口

### 聊天相关

```javascript
// 获取会话列表
GET /api/chat/sessions?userId=1

// 发送消息
POST /api/chat/send
Body: {
  sessionId: "friend_2",
  sessionType: 1,
  receiverId: 2,
  content: "你好",
  msgType: 1
}

// 获取消息历史
GET /api/chat/messages?sessionId=friend_2&limit=50

// 标记已读
POST /api/chat/markRead
Body: {
  sessionId: "friend_2",
  userId: 1
}
```

### 好友相关

```javascript
// 获取好友列表
GET /api/friend/list?userId=1

// 添加好友
POST /api/friend/add
Body: {
  userId: 1,
  friendId: 2,
  remark: "备注名"
}

// 发送好友请求
POST /api/friend/request
Body: {
  senderId: 1,
  receiverId: 2,
  message: "我是 xxx"
}
```

### 群聊相关

```javascript
// 创建群聊
POST /api/group/create
Body: {
  name: "群名称",
  creatorId: 1,
  memberIds: [2, 3, 4]
}
```

### 客服相关

```javascript
// 请求客服
POST /api/service/request
Body: {
  userId: 1,
  serviceType: "general",
  message: "问题描述"
}
```

## 🐛 常见问题

### 1. 消息发送失败
- 检查后端服务是否启动
- 检查数据库连接
- 查看浏览器控制台错误信息

### 2. WebSocket 连接失败
- 检查 WebSocket 配置
- 确保端口 8080 未被占用
- 查看服务器日志

### 3. 会话列表为空
- 检查数据库中是否有数据
- 执行好友系统 SQL 脚本
- 添加测试好友数据

### 4. 样式显示异常
- 清除浏览器缓存
- 确保 CSS 文件正确加载
- 检查浏览器兼容性
