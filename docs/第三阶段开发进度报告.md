## 二、消息类型详解

### 2.1 图片消息

**消息结构**：
```json
{
  "messageType": 2,
  "mediaUrl": "http://.../image.jpg",
  "mediaType": "image/jpeg",
  "fileSize": 102400,
  "width": 1920,
  "height": 1080,
  "format": "jpg",
  "thumbnailUrl": "http://.../thumb.jpg"
}
```

**特点**：
- ✅ 支持缩略图
- ✅ 记录图片尺寸
- ✅ 支持加密传输

---

### 2.2 语音消息

**消息结构**：

## 四、API 接口汇总

### 4.1 文件上传

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/file/upload/image` | POST | 上传图片 |
| `/api/file/upload/audio` | POST | 上传语音 |
| `/api/file/upload/video` | POST | 上传视频 |

### 4.2 消息发送（WebSocket）

| 端点 | 说明 |
|------|------|
| `/app/chat.sendImage` | 发送图片消息 |
| `/app/chat.sendAudio` | 发送语音消息 |
| `/app/chat.sendVideo` | 发送视频消息 |
| `/app/chat.sendEmoji` | 发送表情消息 |
| `/app/chat.sendProduct` | 发送电商卡片 |

### 4.3 消息管理（REST）

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/chat/messages/{sessionId}` | GET | 获取消息列表 |
| `/api/chat/messages/{messageId}/recall` | POST | 撤回消息 |
| `/api/chat/messages/{messageId}` | DELETE | 删除消息 |
| `/api/chat/messages/{messageId}/read` | POST | 标记已读 |
