## 🛡️ 三、安全性和并发控制

### 1. 防止重复提交 ✅
- **实现方式**：Redis + 自定义注解
- **使用示例**：
  ```java
  @PreventDuplicate(interval = 1000, message = "操作过于频繁")
  @PostMapping("/send")
  public Result sendMessage() { ... }
  ```
- **拦截器**：`PreventDuplicateInterceptor`
- **配置**：`WebSecurityConfig`

### 2. API 限流保护 ✅
- **实现方式**：Redis 令牌桶算法
- **工具类**：`RateLimiter`
- **使用示例**：
  ```java
  rateLimiter.isAllowed("api_key", 10, 60) // 60 秒内最多 10 次
  ```

### 3. 数据加密 ✅
- **密码加密**：MD5 + Salt
- **敏感数据**：AES 加密
- **工具类**：`EncryptionUtil`
- **支持算法**：MD5, SHA-256, AES

### 4. 数据库安全 ✅
- **数据脱敏**：视图自动脱敏
- **访问审计**：记录所有敏感数据访问
- **加密日志**：记录加密解密操作
- **访问控制**：基于角色的访问策略

### 5. 并发控制 ✅
- **数据库事务**：`@Transactional`
- **乐观锁**：版本号控制
- **分布式锁**：Redis 锁
